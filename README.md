# 🎓 Student Management System

A full-stack Student Management System built using React and Spring Boot.  
This system allows efficient management of student records, attendance, and academic results with support for large datasets.

---

## 🚀 Features

- 👨‍🎓 Add, Edit, Delete Student Records
- 🔍 Search Students by Name
- 📊 Dashboard with Charts (Marks & Attendance)
- 📋 Attendance Management with Percentage
- 📈 Results Section with Total Marks
- ⚡ Pagination (Handles 5000+ Records Efficiently)
- 🔄 Real-time Data Updates
- 🔐 Basic Login System
- 🎨 Modern UI with Animations

---

## 🛠 Tech Stack

### Frontend
- React.js
- CSS (Custom Styling)
- Chart.js (for data visualization)

### Backend
- Spring Boot
- REST APIs
- Maven
